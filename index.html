Estrutura Básica HTML

<- Voltar

Elementos básicos são o que constituem a base de todo documento HTML. Se vê esses elementos no código fonte de todas as páginas da web, logo em seguida da declaração de doctype, que está na primeira linha da página. Dessa forma, para que um documento seja interpretado pelo navegador, é necessário que o arquivo tenha a extensão .html e a partir disso, poderá ser exibido por qualquer navegador web.

A estrutura básica HTML consiste em basicamente 3 partes:

< html>< /html>

<- Estrutura Básica HTML

Indica que tudo que está dentro dessa tag é HTML.O elemento HTML representa a raiz de um documento HTML, portanto, também é chamado de elemento raiz

< head>< /head>

<- Estrutura Básica HTML

A tag < head> contém coisas que serão carregadas antes da página ser exibido para o usuário. O elemento HTML contém informações legíveis por máquina ( metadados ) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

< body>< /body>

<- Estrutura Básica HTML

O elemento < body> HTML representa o conteúdo de um documento HTML. Tudo que estiver dentro dessa tag é o que de fato vai aparecer para o usuário.


TAGs

<- Voltar

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/).


Atributos TAGs

<- Voltar

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…


TAGs de Texto

<- Voltar

As tags de texto definem textos, estilos de fonte, parágrafos, spans, quebras de linhas, etc. Algumas delas são:


TAG < h1> - < h6>

<- TAGs de Texto

Os elementos HTML < h1>–< h6> representam níveis de título de seção. < h1> é o nível de seção mais alto e < h6> é o mais baixo. Exemplo:

Cabeçalho nível 1

Cabeçalho nível 2

Cabeçalho nível 3

Cabeçalho nível 4

Cabeçalho nível 5
Cabeçalho nível 6

TAG < p>

<- TAGs de Texto

O elemento HTML < p> representa um parágrafo. Em letras visuais, separados por letras, como os blocos identificados em branco. Já em HTML, os parágrafos são para agrupadores de conteúdos e relacionados de qualquer tipo, como imagens de um campo.


TAG < strong>

<- TAGs de Texto

A < strong>tag é usada para definir o texto com forte importância. O conteúdo dentro é normalmente exibido em negrito .


TAG < i>

<- TAGs de Texto

O elemento HTML < i> representa uma parte do texto que é da restante por algum motivo, por exemplo, termos técnicos, expressões destacadas de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico" .


TAG < u>

<- TAGs de Texto

O element0< u> "Underline Element" ou sublinhado é utilizado para sublinhar um espaço definido de um texto.


TAG < mark>

<- TAGs de Texto

A < mark> tag define o texto que deve ser marcado ou destacado.


TAG < blockquote>

<- TAGs de Texto

O Elemento HTML < blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.

Exemplo:

Esta é uma citação tirada da Mozilla Developer Center.


TAG < sup>

<- TAGs de Texto

A < sup> tag define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, exemplo [*].

*Exemplo de uso da tag < sup>.


TAG < sub>

<- TAGs de Texto

A < sub> tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2O.
